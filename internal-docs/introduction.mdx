# Internal Docs

This section is for A5SYNC internal engineers. It describes our architecture, caching implementation, past memory leaks, monitoring setups, and standards.

## How to add documents

Place any engineering specifications in `internal-docs/` and link them in `docs.json`. **Do not blindly copy** MD files from other repos without validating and cleaning them up for Mintlify presentation first!
